Privacy Policy

Chain Reaction by Kaiross • Last updated: December 5, 2024

Summary: Chain Reaction collects minimal data necessary for online multiplayer functionality. We do not sell your data or use it for advertising.

1. Introduction

Welcome to Chain Reaction ("the Game"), developed by Kaiross. This Privacy Policy explains how we collect, use, and protect your information when you use our mobile application and web game.

2. Information We Collect

2.1 Anonymous Authentication

When you play online multiplayer, we use Firebase Anonymous Authentication to create a temporary user session. This generates:

  • A random anonymous user ID
  • No personal information (name, email, phone) is collected

2.2 Game Data

For online multiplayer functionality, we temporarily store:

  • Game state (board positions, current turn)
  • Room codes for private matches
  • Player connection status

This data is automatically deleted when the game ends or after 24 hours of inactivity.

2.3 Local Data

The Game stores preferences locally on your device:

  • Sound settings (on/off)
  • Haptic feedback preferences
  • Game statistics (wins, games played)

This data never leaves your device.

3. How We Use Your Information

We use the collected information solely to:

  • Enable real-time online multiplayer gameplay
  • Match players in auto-match mode
  • Maintain game state during matches

4. Data Sharing

We do not sell, trade, or share your information with third parties, except:

  • Firebase (Google): We use Firebase Realtime Database for online multiplayer. Firebase's privacy policy applies to data stored on their servers.

5. Data Security

We implement appropriate security measures:

  • All data transmission uses HTTPS encryption
  • Firebase security rules restrict data access
  • Game data is automatically cleaned up after matches

6. Children's Privacy

Chain Reaction is suitable for all ages and does not knowingly collect personal information from children. Since we only use anonymous authentication, no personal data is collected from any user regardless of age.

7. Your Rights

You have the right to:

  • Play offline without any data collection
  • Clear local game data by uninstalling the app
  • Request deletion of any stored data by contacting us

8. Changes to This Policy

We may update this Privacy Policy occasionally. We will notify users of significant changes through the app or website.

9. Contact Us

If you have questions about this Privacy Policy, please contact us:

  • Email: contact@kaiross.in
  • Website: kaiross.in
